1. AREA FILTER - Changed from EXACT to PARTIAL matching
   File: src/app/api/salons/route.ts
   
   BEFORE: if (area) q = q.eq("area", area);
   AFTER:  if (area) q = q.ilike("area", `%${area}%`);
   
   NOW WORKS:
   - Filter "Bandra" → Finds "Bandra West" ✅
   - Filter "Andheri" → Finds "Lokhandwala, Andheri West" ✅
   - Filter "Juhu" → Finds "Juhu, JVPD Scheme" ✅

2. SERVICE FILTER - Changed to flexible search
   File: src/app/api/salons/route.ts
   
   BEFORE: q = q.contains("amenities", [service]);
   AFTER:  q = q.or(`amenities.cs.{${service}},name.ilike.%${service}%,description.ilike.%${service}%`);
   
   NOW SEARCHES:
   - Amenities array (partial match)
   - Salon name
   - Salon description
   
   NOW WORKS:
   - Search "Hair" → Finds salons with haircut, hair color, hair treatment ✅
   - Search "Bridal" → Finds salons with bridal services ✅
   - Search "Facial" → Finds salons with facial services ✅
   - Search "HydraFacial" → Finds salons with HydraFacial ✅
